Lists

There are three kinds of lists: ordered, unordered, and definition. List items are wrapped in a list container.

Ordered List

  1. Item one
  2. Second item
  3. Another item
  4. Last one

Unordered List

  • Not again

Links

Links use the a tag and display inline. They must have an href attribute and can use target to open in a new tab.

Links have a different psuedo-class states, if they're hovered or have been visited, for example.

Images

Images use the img tag, which is self-closing, and they might be inside a figure tag. Imagees use the src= attribute to point to the source, which could be anywhere. Images display inline by default.
